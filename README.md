# cyruspahlavi.com

Personal website for Cyrus Pahlavi, Founding Chair of MIGA DAO.

## Deployment

This is a static site - deploy to GitHub Pages:

1. Push to a GitHub repository
2. Go to Settings → Pages
3. Select "Deploy from a branch" → `main` → `/ (root)`
4. Add custom domain `cyruspahlavi.com`

Or deploy to Vercel/Netlify by connecting the repository.

## Structure

```
/
├── index.html      # Main page
├── CNAME           # Custom domain for GitHub Pages
├── images/
│   ├── cyrus-pahlavi.jpg
│   └── favicon.png
└── README.md
```

## Adding Images

Add a portrait photo as `images/cyrus-pahlavi.jpg` (square, at least 400x400px recommended).
